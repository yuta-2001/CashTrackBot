# お金貸し借り管理 LINE BOT

## アプリケーション概要
LINEのBOTを利用して、お金の貸し借りを記録します。
相手を作成し、その相手に対する貸し借り記録をつけることができます。

## 使用技術
### サーバーサイド
- PHP (Laravel)

### フロントエンド
- LIFF (LINE FRONTEND FRAMEWORK)
- HTML/CSS
- JavaScript
- tailwind css => 別リポジトリ

- Next.js => 別リポジトリ

### データベース・インフラ
